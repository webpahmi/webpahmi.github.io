empety-cells

(CSS : Properties)

Properti empety-cells di CSS memilih sel tabel kosong untuk tujuan menentukan apakah akan menampilkan batas dan latar belakang pada mereka. Dengan kata lain, ini memberitahu browser apakah menggambar batas di sekitar sel tabel atau mengisi latar belakang ketika sel itu tidak mengandung konten. Ini seperti menerapkan properti visibilitas pada sel tabel kosong.

# Syntax

CSS
table {

empety-cells : show ;

empety-cells : hidden ;

}

Properti sel kosong memiliki dua nilai utama :

  • show : menampilkan batas dan latar belakang pada sel kosong
  • hide : tidak menampilkan batas atau latar belakang pada sel kosong.

background

(CSS : Properties)

Properti background di CSS memungkinkan Anda untuk mengontrol latar belakang elemen apa pun (apa yang melukis di bawah konten di elemen itu). Ini adalah properti shorthand, yang berarti memungkinkan Anda untuk menulis apa yang akan menjadi beberapa properti CSS dalam satu. Seperti ini :

CSS
body {

background :

url(header.jpg) /* gambar */

top center / 200px 20px /* position / size */

no-repeat /* pengulangan gambar */

fixed /* lampiran */

padding-box /* keaslian */

content-box /* klip */

blue ; /* warna */

}

latar belakang terdiri dari delapan properti lainnya :

  • background-image
  • background-position
  • background-size
  • background-repeat
  • background-attachment
  • background-origin
  • background-clip
  • background-color

289/5000 Anda dapat menggunakan kombinasi properti ini yang Anda sukai, di hampir semua urutan (meskipun urutan yang disarankan dalam spesifikasi di atas). Namun ada gotcha sekalipun: apa pun yang tidak Anda tentukan di properti latar diatur secara otomatis ke default. Jadi jika Anda melakukan sesuatu seperti ini:

CSS
body {

background-color : blue ;

background : url (header.jpg) ;

}

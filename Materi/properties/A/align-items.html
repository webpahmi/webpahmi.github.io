align-items

(CSS : Properties)

Properti align-items adalah sub-properti dari modul Tata Letak Kotak Fleksibel.

Ini mendefinisikan perilaku default untuk bagaimana item fleksibel diletakkan di sepanjang sumbu silang pada baris saat ini. Anda dapat menganggapnya sebagai versi justifikasi-konten untuk sumbu-silang (tegak lurus terhadap sumbu-utama).

Properti align-items menerima 5 nilai yang berbeda:

  • flex-start : batas mulai sumbu awal tepi dari item ditempatkan pada garis awal
  • flex-end : batas mulai sumbu awal tepi dari item ditempatkan pada garis akhir.
  • center : item dipusatkan pada sumbu silang.
  • baseline :item disejajarkan seperti garis belakang mereka disejajarkan.
  • stretch (default) : regangkan untuk mengisi wadah (masih mengikuti lebar minimum / lebar maks).

# Syntax

CSS

align-content : flex-start | flex-end | center | baseline | stretch

.flex-container {

align-items : flex-start ;

align-items : flex-end ;

align-items : center ;

align-items : baseline ;

align-items : stretch ;

}

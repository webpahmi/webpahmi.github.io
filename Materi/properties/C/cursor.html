cursor

(CSS : Properties)

Properti kursor di CSS mengontrol bagaimana kursor mouse akan terlihat ketika itu terletak di atas elemen di mana properti ini diatur. Jelas, itu hanya relevan di browser / sistem operasi di mana ada mouse dan kursor. Mereka pada dasarnya digunakan untuk UX - karena mereka menyampaikan gagasan fungsi tertentu. Jadi cobalah untuk tidak mematahkan keterjangkauan.

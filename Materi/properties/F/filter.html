filter

(CSS : Properties)

CSS filter adalah alat yang ampuh yang dapat digunakan penulis untuk mencapai berbagai efek visual (semacam filter Photoshop untuk browser). Properti filter CSS menyediakan akses ke efek seperti keburaman atau pergeseran warna pada rendering elemen sebelum elemen ditampilkan. Filter biasanya digunakan untuk menyesuaikan rendering gambar, latar belakang, atau perbatasan. Sintaksnya adalah seperti berikut :

# Syntax

CSS
.filter-all {

filter : value (Nilainya) | none ;

}

Where is one of value :

  • blur()
  • brightness()
  • contrast()
  • drop-shadow()
  • grayscale()
  • hue-rotate()
  • invert()
  • opacity()
  • saturate()
  • sepia()
